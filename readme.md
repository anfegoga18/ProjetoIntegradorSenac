## 📖 Descrição  

O aplicativo deverá mostrar uma tela inicial onde o usuário poderá consultar os preços dos produtos digitados. Serão apresentados a ele os produtos selecionados na busca com o menor preço de venda cadastrado na base de dados do aplicativo, estabelecimento e a data de validade do produto.  
Eu, como administrador do aplicativo devo conseguir cadastrar os produtos com os seguintes atributos: nome do produto, tipo de produto, marca, estabelecimento, valor por unidade de volume ou peso (ex: gr/kg/lt/ud por 1 Real pago), tele-entrega do estabelecimento e data do cadastro do produto. Este último atributo é importante porque visa-se criar um histórico dos preços dos produtos e eles variam com o tempo.

## 🛠 Tecnologias utilizadas  

O projeto foi criado com [Spring Initializr](https://start.spring.io/)
- **Spring Boot** (Framework Java para aplicações web)
- **Spring Data JPA** (Integração com o banco de dados relacional)
- **MySQL Driver** (Conexão com banco de dados MySQL)
- **Spring Web** (Criação de APIs REST)
- **Thymeleaf** (Template engine para renderização de páginas HTML dinâmicas)
- **Lombok** (Redução de código com anotações como @Data, @Getter, @Setter)

## ⚙ Configuração de Ambiente  
---

### 1. Pré-requisitos ✅  

- Java 17 (Ou superior)
- Apache NetBeans 22 (Ou superior)
- MySql Server

### 2. Configuração de banco de dados 🗄  

Crie o banco de dados com o comando a seguir:  
```sh
CREATE DATABASE Mercado;
```  
